Задание:
Написать программу, которая из имеющегося массива строк формирует новый массив из строк, длина которых меньше, либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. 
При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

Алгоритм:
Ввод двух массивов: 1 - исходный, 2 - для подстчета результата. 
Обьявляется метод, в методе проходим по элементам первого массива, сравниваем их длинну с заданным числом 3, при выполнении условия
элемент первого массива переходит во второй массив. 
После проверки всех элементов выход из цикла, далее печать второго массива и завершение программы.
